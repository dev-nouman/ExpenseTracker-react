# 💸 Expense Tracker (React)

A modern **Expense Tracker web application** built with React that helps users manage income and expenses with real-time updates, persistent storage, and visual insights.

⚡ Developed as a **solo project** to strengthen React fundamentals and frontend development skills.

---

## 🌐 Live Demo

🔗 http://expense-tracker-react-n.vercel.app/

---

## 🚀 Features

* **➕ Add Transactions** – Easily log income and expenses
* **➖ Delete Transactions** – Remove incorrect or outdated entries
* **💰 Real-time Calculations** – Automatically updates balance, income, and expenses
* **📜 Transaction History** – View a complete list of past transactions
* **💾 Local Storage Integration** – Data persists after page refresh
* **📊 Interactive Visualization** – Dynamic pie chart using Chart.js with category color tags
* **⚡ Dynamic UI** – Fast updates powered by React state and hooks
* **🎯 Responsive Design** – Clean, modern interface across devices

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Functional Components & Hooks)
* **Language:** JavaScript (ES6+)
* **Styling:** CSS3
* **Data Visualization:** Chart.js
* **Data Persistence:** Browser Local Storage API
* **Build Tool:** Vite

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/dev-nouman/ExpenseTracker-react.git
cd ExpenseTracker-react
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in your browser:

```
http://localhost:5173
```
